# Aplikasi-Chat-DBChat
Ini aplikasi chatting Android yang saya buat bernama DBChat.

<p align="center">
  <img src="https://github.com/dioob/Aplikasi-Chat-DBChat/blob/master/Screenshot_446.png">
</p>


<p align="justify">Aplikasi chatting ini saya buat dengan fitur pemanfaatan GPS untuk melacak dan menampilkan lokasi pengguna terkini dengan menambahkan text lokasi dibawah tiap masing-masing pesan chat. Namun demikian pengguna bebas memilih ingin menggunakan maupun tidak menggunakan fitur ini untuk menjaga privasi.<p>

<p align="center">
  <img width="340" height="600" src="https://github.com/dioob/Aplikasi-Chat-DBChat/blob/master/Screenshot_434.png">
</p>

<p align="justify">Fitur ini memanfaatkan GPS untuk penerimaan dan pelacakan lokasi sementara untuk konversi format kode lokasi dan penampil lokasi menggunakan Geocode.<p>

<p align="justify">Aplikasi ini masih dalam tahap pengembangan dikarenakan belum adanya layanan lupa password, belum terintegrasi dengan email asli, tampilan yang masih sederhana dan media penyimpanan yang masih terbatas. Untuk kedepannya bisa dikembangkan untuk publik, maupun private seperti untuk keorganisasian dan lain-lain.<p>
